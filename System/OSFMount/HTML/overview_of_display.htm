Overview of the display

    

 

Menu

The menu bar at the top of the main window provides most of the options for using OSFMount. See Main Menu Options for information about the different configuration options available for OSFMount.

 

List of mounted virtual disks

Shows a list view of all of the mounted virtual disks, including the drive volume letter, image file directory and file name, image size, mounted image properties and the file system type (both detected and reported by Windows). Press F5 to refresh the list.

 

Mount new ...

Opens the Mount image window.

 

Dismount

Dismounts the selected disk.

 

Dismount all & Exit

Force dismounts all virtual disks before exiting.

 

Exit

Exits the application without dismounting any virtual disks.
