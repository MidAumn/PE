Main Menu Options

    

File

Mount new virtual disk...

Open the Mount image window (See Mount New Virtual Disk for a description of mounting options).

 

Dismount all & Exit

Force dismounts all virtual disks before exiting.

 

Exit

Closes OSFMount. Virtual disks shall remain mounted until they are dismounted by OSFMount or the system is rebooted.

 

 

Drive actions

Open

Opens the selected drive using Windows Explorer.

 

Dismount

Dismounts the selected drive.

 

Save to image file...

Save the selected drive to a supported image format. This option is useful for converting from one image format to another, or saving the contents of a RAM disk.

 

Extend size...

Extend the size of the selected drive. The drive must be mounted as writeable to use this option. This option is only supported for RAW image formats. See Supported Image Formats for more details.

 

This option will change the original disk image file and as such, is not recommended for forensic analysis of disk images.

 

Format...

Format the selected drive. The drive must be mounted as writeable to use this option.

 

This option will destroy the contents of the original disk image file and as such is not recommended for forensic analysis.

 

Set media read-only

Change the selected drive to be read-only. When this option is selected the contents of the original disk image file is not changed and as such is the recommended option for Forensic analysis. This is the default behavior.

 

Set media writable

Change the selected drive to be writeable. When this option is selected the contents of the original disk image file may be changed since Windows may write to the image file without any explicit user access to the disk. This may remove useful forensic information from the disk image file (such as overwriting deleted NTFS Master File Table entries).

 

This option is not recommended for forensics use.

 

Set media removable

Change the selected drive to be removable. This changes the disk caching behavior of the drive.

 

Set media fixed

Change the selected drive to be a fixed disk. This changes the disk caching behavior of the drive.

 

This option is default and recommended for forensics use.

 

 

Help

Help

Opens this help file. Context related help is available from most windows using the “F1” key.

 

PassMark on the web

Opens the default browser at the PassMark website home page.

 

About

Displays the version and copyright information about the currently installed version of OSFMount.
