System Requirements

    

Operating System:

•Windows XP, Vista & Win 7 (32-bit/64-bit)
•Windows Server 2000, 2003, 2008 (32-bit/64-bit)

 

For 64-bit Windows operating systems, the 64-bit binaries must be used.

Users must have administrator privileges.

 

RAM:

•128MB

When mounting large disk images and/or mounting images to memory, the more RAM the better.

 

Disk space

•1 MB of free hard disk space for the installation files.
